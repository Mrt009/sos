Disaster SOS Offline Demo

Works offline after first load. Calls and SMS use cellular network.

1) First-Time Setup

Save numbers for each category. If category is unknown, fallback number is used.

2) Emergency Action

Tap category, then app opens call and prefilled SMS. User confirms on phone UI.

Notes

  • First install/open requires internet once.
  • After that, app UI can run without internet.
  • Call/SMS still require cellular signal.
